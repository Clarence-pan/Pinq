Pinq\ITraversable

interface ITraversable implements IAggregatable, IteratorAggregate, ArrayAccess

The root interface providing a fluent query API for a range of key value pairs.

Query calls must be immutable and return a new instance with every query call. Queries should also be executed lazily upon iteration.
