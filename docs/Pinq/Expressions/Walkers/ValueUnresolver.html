Pinq\Expressions\Walkers\ValueUnresolver

class ValueUnresolver extends ExpressionWalker

Unresolves and stores any values in the expression tree and replaces them with uncolliding variable names.

This is useful when compiling an expression tree and certain values shouldn't be stored inline, such as objects or arrays.

3 + $Var; === will become === ${'some unclashing name'} + $Var
